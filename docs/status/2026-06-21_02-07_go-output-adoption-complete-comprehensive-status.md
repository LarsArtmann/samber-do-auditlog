## B. PARTIALLY DONE ⚠️

### B.1 Diagram theming after go-output migration

The warm-amber palette is preserved **per-node** via `GraphStyle{Fill, Stroke, FontColor}`. However two visual elements were lost in the migration because go-output renderers lack graph-level attribute setters:

- **DOT dark background** (`bgcolor="#14110d"`) — gone. go-output's `DOTRenderer` exposes `SetRankDir`/`SetSplines`/`SetNodeSep`/`SetRankSep` but no graph-bgcolor/fontcolor setter. Node fills (amber) remain.
- **Edge line-colors** — Mermaid `lineColor` and PlantUML arrow color no longer emitted. Renderers style nodes, not edges.

**Status**: Nodes look right; the dark canvas + amber edges do not. Path to restore = upstream contribution to go-output (graph-attributes setter).
